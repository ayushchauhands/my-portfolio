Sofware Development Lifecycle

What is Software?

Software itsels is the set of instructions or programs that tell a computer what to do.

It is independent of harware and makes computers programmable.

What is Sofware Development?

Software Development refers to a set of computer science activities dedicated to the process of creating, designing, deploying and supporting software.

Types of Software

System Software

System software is a set of programs that controls and manage the operations of a computer hardware. It also helps application programs to execute correctly.

e.g. Operating System, Device Drivers, Harware Management etc.

Programming Software

Programming sofware is a program or set of programs which helps the software developer by assisting them in creating, debugging and maintaining other programs and applications.

e.g. Eclipse, Wolfram Mathematica, Sublime Text, Java

Application Software

Application software are software created for a specific purpose. It is generally a program or set of programs used by end users.

e.g. Uber, AirBNB, Facebook, Whatsapp etc.
